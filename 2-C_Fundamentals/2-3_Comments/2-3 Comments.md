# 2.3 Comments

```c
/* This is a comment */
```

This type of comment can span multiple lines.

```c
/*
 * Name: pun.c
 * Purpose: Prints a bad pun.
 * Author: K. N. King
*/
```

## C99

C99 has a second kind of comment

```c
// This is a comment
```

This style of comment ends automatically at the end of a line
